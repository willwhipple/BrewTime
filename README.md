# BrewTime

A minimalist iOS companion for an espresso-focused lifestyle — pastel UI, cup counter, and espresso recipe cards (Bloom & Brew spec).

- **Platform:** iOS 18+ (SwiftUI)
- **Features:** Coworking cup counter (`@AppStorage`), horizontal recipe cards (Latte, Flat White, Cappuccino, Cortado), recipe detail sheet with ratio bar, haptics, spring animations, light/dark pastel theme

## Run locally

1. Open `BrewTime.xcodeproj` in Xcode.
2. Select an iOS Simulator (e.g. iPhone 17).
3. Press **⌘R** to build and run.

See [spec.md](spec.md) for the full design spec.
